### Intro

Gensokyo in Memory

### How to use

 1. Install **Minecraft** and **Forge(>=1.12.2-2847)**
 2. Download the latest version of **mirror(>=0.5.0)** & **DanmakuCore(>=0.8)**
 3. Download this mod from the repo
 4. Put them on **mods** directory
 5. Launch Game and Enjoy
